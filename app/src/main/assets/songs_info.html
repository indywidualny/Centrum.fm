Dnia 23 maja 2016 o godzinie 22:46:03 ruszyła automatyczna usługa zbierania danych o piosenkach, pojawiających się na antenie Radia Centrum.

Dostęp do Songs API jest otwarty, jednak najwygodniejszą przeglądarką historii anteny jest niniejsza aplikacja. Moim celem jest nieprzerwane gromadzenie wyemitowanych utworów.

Utwory pobierane są z serwisu Radia Centrum, który niestety ma problem z kodowaniem ogonków. Dopóki problem nie zostanie rozwiązany po stronie rozgłośni, polskie znaki będą wyświetlały się jako uszkodzone.
